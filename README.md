## Setup

```sh
npm install
```

## Usage

```sh
npm run dev # run in development mode
npm run build # build for production
npm run start # run in production mode
```

## Deployment

Auto deploys to Contabo VPS via rsync in github action

Or to push manually

```sh
npm run deploy # deploy to netlify
```
